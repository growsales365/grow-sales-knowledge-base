## WRITTEN GUIDE

### What Is Groq?

Groq is an AI company that runs open-source AI models — including OpenAI's Whisper — blazing fast and mostly free. Think of it as a supercharged API that lets you do things like:

- **Transcribe any audio or video to text** (Whisper)
- **Run AI language models** (LLaMA, Mixtral, Gemma)
- All at speeds 10-100x faster than most competitors

For our purposes, the killer use case is **free YouTube transcription** — no cookies, no browser extensions, works from anywhere including your phone.

### How To Set It Up (5 minutes)

**Step 1: Create a free account**
Go to **console.groq.com** → Sign up (free)

**Step 2: Get your API key**
- Click your profile → API Keys
- Click "Create API Key"
- Copy the key (starts with `gsk_...`)

**Step 3: Send it to MasterClaw**
Just paste it in WhatsApp — I'll save it to the server and wire it up automatically.

**That's it.** After that, just send me any YouTube URL and I'll return the full transcript. No cookies, no extensions, no Mac required.
